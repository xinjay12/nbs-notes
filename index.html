核心 Vision / Mission / Core Values
壽險顧問 的本質
  • LP = Love + Care，陪伴客戶一生的規劃者
  • 壽險顧問即是醫生——把對方當成人，不只是病人
  • 「相信，開口，行動」
  • 因為相信自己的職業極有價值
服務策略 Tier 架構
Tier 1｜加入保險之前
① 向準保戶說明保障的重要性
② 透過需求分析了解客戶需求（第一次碰面）
③ 提供最佳解決方案，完成銷售（第二次碰面）
Tier 2｜售後服務
定期與客戶聯絡 · 具備解決問題的能力
週年檢視 · 不可 over promise · 不要不懂裝懂
心態關鍵
  • 反對意見是理所當然的，拒絕是正常的
  • 沒有成績，什麼都是假的
  • Mindset is First：自信是包裝出來的
  • 撞牆時別繞過去——攀過去
  • 成功是自己體驗到的過程，成就是給別人看的

SPIN 問法
S
Situation 情境性
事實、情況，已發生的；了解背景基本資訊
P
Problem 探索性
了解情況背後的動機（Why / Who / When / Where / What / How）
I
Implication 暗示性
尚未發生的問題，引導對方感受潛在風險
N
Need-payoff 解決性
讓客戶說出需要解決的問題，喚起解決意願
關鍵問句
「假設明天身故了，您的家人會如何運用這筆保險金？」
  • Ask the man：小名、個性、興趣、為何推薦、已婚未婚？
  • OI 可多人，所以一定要個人（每人想法不同）

家庭需求分析 FF
三大風險框架
⚡
走得太早
死亡保障
LNB · 分期給付
身故：把握一切都 cover
⏳
活得太久
年金 · 退休規劃
帳戶價值持續累計
→ 保價金
🔗
走不了
殘廢 · 長照
完全失能 / 7 種狀況
Long Term Care
費用結構（必要支出）
  • 生活費：食衣住行，基本日常
  • 教育費：學業、才藝、補習
  • 房貸：水電瓦斯、管理費
  • 父母奉養金
  • 醫療：看護、手術、藥品、病房
  • 退休準備
最先問、因最清楚
生命的經濟價值
  • 薪水 × 工作年數 = 生命的經濟價值
  • 保額 = 年收的 10 倍
  • 保費 = 收入的 10%
  • 三口之家費用：2000萬（20年 Total）
「還記得花了這些錢後，如果你走了，家人會拿到多少錢？」
FF 核心原則
  • FF = 現在的生活、未來的生活藍圖
  • 重提 FF 的事實及感覺，不是只有冰冷文字
  • 實際談論客戶的情況（非說教）
  • 問問題要以幫客戶想有限選項（一、兩個），是非題形式來問
  • 情感面的代入 → 真的發生，那畫面是什麼？
05
各族群需求切入
家庭有子族群
  • 說明順序：明天就發生+很嚴重 → 死亡 → 失能 → 提前給付
  • 房貸反對：「孩子還這麼小！>>風險不會等你」
  • 教育費反對：「去打工！>>小孩還這麼小！」
  • 在反對問題前，多詢問原因（為什麼要買房？為何選這裡？）
「假如沒有明天」
人最重要的是時間
單身族群
  • 自願性（有選擇）vs 非自願性（有計劃）
  • 走得太早 → 父母還有多久退休？
  • 活得太久 → 財產 50 年後還能用嗎？→ 年金
  • 走不了 → 誰來照顧你？他們工作怎麼辦？能顧多久？
「如果有得選擇呢？」
醫療需求切入
  • 「不怕死，怕死不了」
  • 在醫院期間不長，出院之後呢？
  • 費用：看護、薪資損失、手術、藥品、病房
  • 健保 → 死亡（基本給付），其餘需自費
反對問題心法
  • 對客戶的反對問題，我都自設「很久以後」「時間很充足」
  • → 人最重要的是時間，假如沒有明天
  • 每個反對問題都要探索背後原因
  • 先稱讚，再溝通（先 Yes but 再說明）
06
建議書說明 P1 / P2
P1 準備原則
  • 檢視 FF 時客戶談到的東西，找缺口
  • 以自己身為保戶的角度，用淺顯易懂的方式傳遞概念
  • Why 壽險業? → 接上次打電話說的家庭保障計劃
  • 用 SPIN 的方法複習 P1 要說明的 6 個核心，釐清背景
P1 說明順序
死亡
身故 · 提前給付 · 分期給付
失能
完全失能 · 帳戶價值累計
老
老年住院 · 長照 LTC
先設計大的費用
WL 的六大價值
  • 子女未來的護身符（保價金：孳息脫費 + 退休金）
  • 保障未來之資產
  • 可靠的資產 = 保價金（可直接動用，不需保證人）
  • 遺產稅源
  • 資產配置
  • 現金作為遺產分配的方法，最無爭議
P2 儲蓄險切入
  • 為何想買？為何想一年存這些？有個目標嗎？
  • FF = 喚起需求 + 原先投保之問題
  • 幫小孩買 → 贈學稅問題（贈與額度計算）
  • 死亡險最低稅賦額 670 萬，超出者稅 20%
07
促成 Close
Close 工具 C.S.E
  • C（Choice）：Can you qualify? Live or Die？製造選擇
  • Seek：尋找真正原因，重新確認，看怎麼調整保費
  • Example：與客戶有關的故事（各項費用）
  • Obligation：喚起責任感
關鍵 Close 話術
「這筆錢如何照顧你在意的人」→ 要敘述、畫面
「如果這是必要的，那為何要用比較貴的成本來買？」
「花多少？你付出的。賠多少？跟你拿到的。划算嗎？」
反對問題處理
  • 反對問題至少要處理 5 次
  • 不安不滿 → OI 確定購買 → P1、P2、Close
  • 為何馬上決定優於延後？
  • ① 符合你當下的情況
  • ② 想買不見得買得到（風險的不可見性）
  • ③ 這個決定是最好的
  • 「你不是很認同嗎？那就當下決定。」
故事 Close
  • 複習法 + 景象（FF 談過的東西）
  • 受益人會面臨的情況
  • 當下完成投保的程序
  • PC → N，說故事是關鍵
「我相信你會這樣能實際照顧家」
08
售後服務與推薦介紹 Ref
Ref 的時機
  • 遞送保單、理賠服務時更好要名單
  • 20-10-千分之5法則：每個階段都能要 Ref
  • 客戶推薦，是來自從第一印象的累積
  • 重視推薦介紹，自己重視，客戶才能重視
要 Ref 的三步驟
① 讚美客戶
真誠稱讚，讓對方開心說話
② 請客戶給意見
哪裡不錯？哪裡可以更好？具體說明
③ 說明原因，提出需求
「因為我想幫台灣家庭了解保障不足的問題，所以需要你推薦結婚、有孩子的朋友」
Ask the Man 要點
  • 小名、個性、興趣、為何推薦此人？已婚未婚？
  • 「用一句話形容此人，會是？」
  • 「可以提到你的名字嗎？」
  • 完全不可提供：推薦人買了什麼、談了什麼
  • 「跟此人聯絡後的情況我會再跟你說！」
After FF 要 Ref
  • 假如有一天 A 身亡，A's wife 找你幫忙：
  • 房貸？Y/N　教育費？Y/N　生活費？Y/N
  • 「時間可以沖淡傷痛，但幫不走這些費用」
  • 不過現在 A 仍健在，你願不願意把他介紹給我？
PICA 服務架構
P - Plan 計劃 I - Information 資訊 C - Continue 持續 A - Awareness 提醒
  • 生日賀卡 + 電話
  • 已結婚的 → 記錄結婚紀念日
  • 週年時的服務：列出有關係者
  • 思考客戶的五同：「我該怎麼問比較好？」

當下約與時間管理
BTP 計劃表
  • FF → PC 愈快愈好（客戶 100% 遺忘）
  • 計劃表：P → P1 大約 15-20 分鐘
  • 複習 FF 表所提過的東西
  • 細心觀察責任感
  • 把大目標拆成小目標
  • 走出舒適圈，即時反饋
當下約的原則
晚出來的班表，就需要約一個聯絡的時間
  • 專業人士的時間管理：當下確認 or 敲定聯絡時間
  • 告知何時會聯絡
  • 要勉強你的客戶
  • 3 天後中午確認可否
每週工作計劃
  • Searching：起商探客（固定時間）
  • Selling：銷售流程（OI→FF→PC→Ref）
  • Service：遞送保單、理賠、保單變更
  • Study：證照考試、商品研究、外聘課程
  • 首月 30 件

銷售心理與溝通技巧
客戶為何買
  • 每個人的開關不同
  • 使他想到別人對他的好
  • 促成決定的 Key：確保未來的生活不會破碎
  • 買東西不是買這種東西本身，是買一個生活形態
  • 「生活即銷售」：Movie / Books / Life
客戶為何不買
  • 基本弱點：拖延、省力、迴避、缺乏決心
  • 為何拖延？可能有更好的 / 在尋求認同
  • 快掛了 — False（借口）
  • 找不到方法 — True → 還有什麼方法沒試過？

溝通核心原則

  • Nobody care how much you know, unless they know how much you care
  • 聽你講是為了要了解，而非回答

客戶想問而沒問的四件事
① 舒適度 ② 信任度 ③ 執行度（你知道你在幹嗎） ④ 誠意
① - ④ 先建立，② - ③ 持續維護

利率變動型
  • 利率型：保額逐年下降、保價金下降、以宣告利率為基礎
  • 美國公債利率逐年降 → 影響宣告利率
  • 全球利率型（標準體）：雙豁免
  • 境外保單：保險金匯回 + 20% 所得；遺產稅一定被課
停效與否（負債的問題）
  • 借的錢花到其他地方
  • 抵押品無增值的機會
  • 借款無法轉換收益空間
  • 利息與回收無投資價值
  • 槓桿比例永遠負大於正
退休資金三分法
  • 40-60 歲：保障工作能力
  • 60 歲以上：長壽風險是其他所有風險之乘數
  • 創造終身保證收入 = 年金
  • Pay checks & Play checks
  • 投資 → 看 Timing（進/出場時機）
  • 理財 → 看 Time（時間紀律，享受長期複利）
